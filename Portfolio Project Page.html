Arcade Game Website

This project allowed me to apply my learning in HTML, CSS, and JavaScript to create a site with six engaging games.

    • Invaders: A classic shooter game.
    • Brick Breaker: A ball-and-paddle game.
    • Dragon Repeller: An adventure game.
    • Flappy Bird: A simple but addictive game.
    • Tic Tac Toe: A two-player strategy game.
    • Math Game: A fun educational game.

I used HTML for structure, CSS for design, and JavaScript for game interactivity, improving my skills in these technologies.
